# DiscordBot_Qutilvil
Дискорд бот на C# с использование DSharpPlus (https://dsharpplus.github.io/index.html)

Команды:

[] - обязательный параметр

!!serveronline (so) - показывает ваш онлайн на сервере с ...

!!online (o) - показывает ваш онлайн в дискорде с ...

(Не работает)!!clist [name_playlist] - создать плейлист

(Не работает)!!alist [name_playlist] [source] - добавить в плейлист

(Не работает)!!plist [name_playlist] - воспроизведение плейлиста

!!changerole [new_name] - изменить роль

!!changecolor [#color] - изменить цвет

!!search [search] - поиск в Youtube

!!playlist [path] - плейлист в Youtube (ссылка)

!!dateentry - первое появление на сервере

!!ava - аватарка?

!!gif [req] - гифка

!!inst [username] - рандомная из 12 первых фоток
